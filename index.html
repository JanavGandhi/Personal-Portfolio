DSA Knowledge

I have mastered these topics in DSA

Stacks

A stack is a data structure that follows the Last In, First Out (LIFO) principle, where elements are added and removed from the same end.

Queues

A queue is a data structure that follows the First In, First Out (FIFO) principle, where elements are added at the rear and removed from the front.

Searching

Searching in data structures is the process of finding the location of a specific element within a collection, such as an array, list, tree, or graph.

Sorting

Sorting is the process of arranging elements in a specific order, typically ascending or descending, to facilitate easier access and analysis.

Linked List

A linked list is a linear data structure where elements, called nodes, are linked using pointers, with each node containing data and a reference to the next node in the sequence.

Trees

A tree is a hierarchical data structure consisting of nodes, with a single root node and sub-nodes forming parent-child relationships, used to represent data with a branching structure.
